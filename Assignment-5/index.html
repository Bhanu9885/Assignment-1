Objective :

Use at-least 7 css selectors to change the style of HTML elements

We use 7 CSS Selectors to style this HTML Page:

S.No Type of Selector Examples Definition
1 Universal Selector * { } This Selector gives properties to whole HTML page.
2 Element Selector h1 { } This Selector gives properties inside h1.
3 Class Selector .container { } This Selector gives properties to whole elements inside the Class=container.
4 ID Selector #container { } This Selector gives properties to whole elements inside the ID=container.
5 Grouping Selector div , span { } This Selector gives Same properties to both div and span elements.
6 Psudo Class Selectors a:hover { } This Selector gives properties to <a> tag when the action hover is done over <a> tag.
7 Child Combinator Selectors ul>li { } This Selector gives properties <li> which are nested directly inside <ul>
